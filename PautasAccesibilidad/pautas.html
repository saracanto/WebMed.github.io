Identificación del Idioma Principal

El atributo lang en el elemento <html> indica que el idioma principal de esta página es el español (lang="es").

Etiquetado Semántico

Esta página utiliza elementos semánticos como <header>, <nav>, <main> , <footer> , <section> y <article>. Esto mejora la navegación y comprensión para usuarios con discapacidades visuales y ayuda a estructurar el contenido de manera clara y comprensible.

Uso de encabezados jerárquicos (h1-h6)

Esta página web proporciona una estructura clara y lógica en el contenido. Esto se logra utilizando correctamente las etiqeutas de encabezado como, <h1>, <h2>, <h3>,etc. Los lectores de pantalla y otros dispositivos asistivos utilizan esta jerarquía para interpretar la estructura de la página.

  • <h1> : Debe ser usado solo para el título principal de la página, que representa el tema general.
  • <h2> y <h3> : Estos encabezados se usan para organizar el contenido en secciones y subsecciones.

Formularios accesibles

Para que un formulario sea accesible, cada campo debe estar claramente etiquetado para que los usuarios con lectores de pantalla puedan entender qué se espera en cada campo. Esto se logra mediante el uso de la etiqueta <label>.

<label> : Debe estar asociado con su campo de entrada utilizando el atributo for, que debe coincidir con el atributo iddel campo correspondiente. Esto asegura que cuando un usuario haga clic o seleccione la etiqueta, el campo de entrada asociado obtenga el foco.

Enlaces descriptivos

Los enlaces deben ser claros y descriptivos, brindando información sobre su destino. Los enlaces genéricos como "haz clic aquí" no son suficientes, ya que no proporcionan un contexto útil a los usuarios con discapacidades.

  • Usa textos que describan la acción o el destino del enlace.
  • Evita los enlaces genéricos, ya que no proporcionan suficiente información.

Mecanismos de Navegación Implementados

Este sitio web proporciona mecanismos de navegación accesibles mediante:

  • Una barra de navegación principal utilizando el elemento <nav>.
  • Migas de pan (<nav aria-label="breadcrumb">) para mostrar la jerarquía de navegación.
  • Enlaces con textos descriptivos y claros.

Implentación de CSS y hojas de estilo accesibles

Este sitio web proporciona múltiples modos de navegación:

  • Modo Noche:El modo noche utiliza colores oscuros para reducir la fatiga visual y la luminosidad de la pantalla en entornos con poca luz. Esto ayuda a disminuir la exposición a la luz azul, lo que puede mejorar la comodidad y el sueño
  • Estilo de Alto Contraste para Baja Visión:Los estilos de alto contraste ayudan a las personas con baja visión a distinguir mejor entre el texto y el fondo. Esto es esencial para la accesibilidad, permitiendo que más usuarios accedan a la información sin esfuerzo adicional.
  • Estilo con Tipo de Letra Mayor:Aumentar el tamaño de la fuente es fundamental para las personas con problemas de visión. Un texto más grande facilita la lectura y la comprensión, asegurando que todos los usuarios puedan acceder al contenido sin problemas.
  • Estilo de Alto Contraste con Tipo de Letra Mayor:Combinar alto contraste y un tipo de letra mayor maximiza la accesibilidad para personas con baja visión. Esto permite que quienes tienen dificultades para ver tanto el contraste como el tamaño del texto puedan leer y entender la información sin esfuerzo adicional.
